# Wreck-It Ralph Fan Page

This repository contains a simple HTML and CSS practice project featuring a fan page for the animated movie character Wreck-It Ralph.

## Files

* `index.html`: The main fan page for Wreck-It Ralph, including information about the character and his friends. It uses a custom Google Font ('Press Start 2P') to give it an arcade-style look.
* `css/hero.css`: The stylesheet for the main fan page.
* `style.html` & `css/test.css`: A simple test page to practice and experiment with various CSS styling rules, including basic typography and color testing.
* `test.html`: Another basic HTML test page demonstrating headings, paragraphs, links, and image embedding.

## How to View

To view the project, simply open `index.html` in any modern web browser. You can also open `style.html` or `test.html` to view the practice pages.
